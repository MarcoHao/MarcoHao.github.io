# **多智能体论文**  
*Research on Multi-Agent Structural Generation and Optimization Based on Hierarchical Behavioral Control*

### ——从结构理性到装配逻辑的智能协同建构

> Multi-Agent System, Python, Grasshopper

**论文研究项目 · 2025**  
东南大学建筑学院 · 指导教师：王大嵩、唐芃

本研究聚焦建筑结构生成与优化的一体化问题，提出一种基于分层行为控制的多智能体系统（Multi-Agent System, MAS）框架，旨在在设计早期阶段协同考虑结构性能、构件可装配性与几何逻辑。通过“架构—机制—策略—行为”四层结构，将复杂的生成任务从全局调度逐层分解至局部建构行为，实现设计逻辑与建造逻辑的统一。

关键词：计算性设计、生成式优化、多智能体系统、结构建构一体化

### 论文摘要｜Abstract

随着计算性设计在建筑全生命周期中的深入应用，如何在设计阶段同步满足结构合理性与装配可行性，并有效协调多重约束，已成为数字建造领域的关键挑战。当前主流方法多依赖基于规则的生成流程或基于样本池的后验优化，难以对构件行为、结构理性与几何需求之间的动态协同进行系统建模。

本文提出一种分层多智能体生成与优化框架，以“架构—机制—策略—行为”四层视角组织系统设计：  
- **架构层**定义全局调度智能体与局部建构智能体的职责分工；  
- **机制层**通过“全局引导—局部响应”的控制逻辑协调生成与评估；  
- **策略层**采用分阶段建构策略，逐步引入约束；  
- **行为层**基于“感知—决策—反馈”循环执行局部动作并消解冲突。  

为连接离散组合空间与连续优化器，研究引入组合数映射方法，将离散初始点位编码为连续索引，显著提升搜索效率。

以自由曲面钢框架为案例，在 PyCharm 与 Rhino 工具链中开展生成与优化实验。结果表明，分层行为引导有效降低了智能体行为的离散性，提升了结构性能、构件合格率与曲面覆盖率，同时在保持方案多样性的同时增强了收敛稳定性。该框架验证了多智能体系统在设计—建构一体化中的可行性，为未来智能协同建造提供了新路径。

实验进一步表明：分层控制机制显著增强了智能体间的协同性，结构整体性能提升，构件合格率与曲面覆盖率均取得明显改善。系统在维持解空间多样性的同时，实现了更高的生成效率与优化稳定性。

本研究从系统行为视角重新定义建筑生成过程，提出了一条兼顾全局逻辑与局部自治的设计建构一体化路径，展现了在生成逻辑建模、算法架构设计、以及 Rhino + Python 跨平台集成方面的综合能力。
